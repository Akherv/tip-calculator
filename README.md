# Projet React + Vite

Ce projet utilise **Vite** pour le bundling et le dev server, avec **React**.

## Prérequis

- Node.js **>= 18**
- npm / yarn / pnpm

Vérifier la version de Node :

```bash
node -v
```

## Installation

```bash
Vite

```

Ouvrir :
http://localhost:5173/
